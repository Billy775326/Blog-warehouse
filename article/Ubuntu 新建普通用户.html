创建一个新用户：

sudo adduser tempuser

然后按照提示输入新用户的密码和其他信息。

为新用户添加必要的权限：

如果安装程序需要特定的权限，你可以将新用户添加到 sudo 组中，但在运行安装程序时不要使用 sudo。

sudo usermod -aG sudo tempuser

切换到新用户：

su - tempuser

运行安装程序：

在新用户的环境中运行安装程序，不使用 sudo。

./ABC.run

完成安装后删除临时用户（可选）：

如果不再需要临时用户，可以将其删除。

sudo deluser tempuser --remove-home

这样可以避免安装程序因为使用 root 或 sudo 而出现的错误。如果有需要在安装过程中提升权限的情况，可以手动处理或在适当时候切换回 root。
